Listas

HTML ofrece a los desarrolladores varios mecanismos para especificar listas de información. Todas las listas deben contener uno o más elementos de lista:

  • Información desordenada.
  • Información ordenada.
  • Definiciones.

Ejemplo 1:

Una lista desordenada comienza con la etiqueta "ul". Cada elemento de la lista comienza con la etiqueta "li". Los elementos de la lista están marcados con viñetas, es decir, pequeños círculos negros de forma predeterminada.

                
                 <ul>
                  <li>Data Structures & Algorithm</li>
                  <li>Web Technology</li>
                  <li>Aptitude & Logical Reasoning</li>
                  <li>Programming Languages</li>
                 </ul>

Ejemplo 2:

Una lista ordenada comienza con la etiqueta "ol". Cada elemento de la lista comienza con la etiqueta "li". Los elementos de la lista están marcados con números de forma predeterminada.

                
                 <ol>
                  <li>Bread</li>
                  <li>Eggs</li>
                  <li>Milk</li>
                 </ol>

Resultado:

Por defecto las listas ordenadas utilizan numeros para listar los elementos, en caso de querer un comportamiento diferente por ejemplo letras entonces puede hacer lo siguiente:

                
                 <ol type="A">
                  <li>Bread</li>
                  <li>Eggs</li>
                  <li>Milk</li>
                 </ol>
